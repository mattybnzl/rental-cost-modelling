# The Cost of a Roof — AU/NZ Rental Cost Model

An interactive model for exploring rental costs across Australia and New Zealand, with adjustable parameters for market conditions, policy scenarios, and regional comparison.

**[View the live tool](https://mattybnzl.github.io/rental-cost-modelling/)**

## About

This is a single-page interactive HTML tool — no install required. Click the link above to use it in your browser.
